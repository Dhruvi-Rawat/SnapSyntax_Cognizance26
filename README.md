# 🚀 SnapSyntax Cognizance 2026

## 📌 Overview
This project is a frontend webpage layout developed as part of the SnapSyntax competition at Cognizance 2026 (IIT Roorkee).

The objective was to recreate a given UI design using only HTML and CSS within a limited time frame, focusing on layout accuracy, clean structure, and efficient styling.

---

## 🎯 Objective
- Replicate the provided UI design as closely as possible  
- Implement a structured and clean layout  
- Practice CSS positioning and layout techniques  
- Build efficiently under time constraints  

---

## 🛠️ Tech Stack
- HTML5  
- CSS

---

## ✨ Features
- Responsive and clean webpage layout  
- Top navigation bar for easy navigation  
- Profile section aligned to the top-left  
- Projects section aligned to the top-right  
- Centered container for main content  
- Minimal and structured UI design  

---

## 🧠 Approach
- Designed the layout structure using semantic HTML  
- Used CSS Flexbox for alignment and positioning  
- Focused on maintaining visual balance and spacing  
- Ensured clean and readable code structure  

---

## ⚡ Challenges Faced
- Achieving proper alignment across sections  
- Managing spacing and positioning effectively  
- Recreating the design within limited time  

---

## 📚 Key Learnings
- Strong understanding of layout design principles  
- Improved skills in CSS positioning and Flexbox  
- Learned to work efficiently under time pressure  
- Better approach towards structuring frontend code  

---


## ⭐ Conclusion
This project reflects my ability to design and implement structured webpage layouts using HTML and CSS, while maintaining clarity, simplicity, and efficiency.
